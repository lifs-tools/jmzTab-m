About MzTab-M 2.0 reference implementation.

A reader, writer and validator reference implementation for the mzTab format for metabolomics from version 2.0 onwards.

Project Modules

This project has declared the following modules:

Name Description
jmztabm-api The data model for the mzTab-M 2.0 data standard format for metabolomics.
jmztabm-io Reading, writing and basic validation of mzTab-M data files in JSON, CSV or object format.
jmztabm-cli The command line interface for reading, validation and conversion of mzTab-M 2.0 JSON and CSV files
jmztabm-validation Semantic validation with rule mapping and controlled vocabular terms
jmztabm-test-utils Utilities for extracting and testing of mzTab-M test files.
